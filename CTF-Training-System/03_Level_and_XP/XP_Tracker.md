## Calculation Notes – Bandit Levels 0–20

**XP Rules Used:**

- Easy Challenge Solved: 50 XP  
- Writeup Created: 100 XP per writeup  
- Cheat Sheet Updated: 5 XP per update  
- Timed Practice: 5 XP per session

### Notes

- Each writeup counts individually toward XP even if multiple were done the same day.  
- Cheat sheet and timed practice XP are counted per day/session, not per challenge.  
- Easy challenges are counted separately from writeups (solving + documenting both give XP).  
- This calculation assumes all levels are “Easy” difficulty in Bandit.
