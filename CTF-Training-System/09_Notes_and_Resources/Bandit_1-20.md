Important Concepts Learned
•	SSH login and authentication methods (password vs private key)
•	Linux file system navigation (absolute vs relative paths)
•	Hidden files and directories
•	Permissions (read/write/execute) and their importance for keys/scripts
•	Searching files with find and grep
•	Decoding files: Base64, ROT13, hexdump
•	Temporary directories for safe file manipulation
•	Port scanning and identifying SSL/TLS services
•	Using diff to detect changes between files
•	Executing binaries and setuid programs

Tips for Beginners
•	Always save passwords and keys locally; Bandit does not remember them.
•	Work step by step, don’t rush.
•	Use temporary directories when manipulating files to avoid permission issues.
•	Check file permissions if private keys or scripts fail to run.
•	Review and note every command you used; it helps in later levels.
