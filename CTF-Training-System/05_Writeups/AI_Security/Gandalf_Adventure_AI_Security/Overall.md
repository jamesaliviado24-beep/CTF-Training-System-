## Key Security Concepts Learned

- **Narrative bypass** — story framing lowers prohibition enforcement
- **Indirect speculation** — "what would someone else say" extracts without direct disclosure
- **Multi-character bypass** — distributing disclosure across characters evades per-character rules
- **Symbolic framing** — treating the forbidden topic as a symbolic object bypasses direct naming rules
- **Hard rule injection** — `###` markers override existing model instructions
- **Compliance reframing** — making disclosure appear to be rule-following rather than violation
